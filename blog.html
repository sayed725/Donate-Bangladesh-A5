IMPORTANT FAQ

What is the Document Object Model (DOM)?

DOM is the structural pieces of a web document which holds all of the HTML elements and their properties. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

You can an element from the DOM by using: getElementById(), getElementsByClassName() .getElementsByTagName(),querySelector(), querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is the method of handling events for multiple elements by an event listener on one parent element.It allows events triggering on child elements to be captured and handled by parent. It is a useful pattern which allows us to write cleaner code, and create fewer event listeners with similar logic.

How do you manipulate an element's attributes and styles using the DOM?

By using document.getElementById or querySelector you have to access the elements then using setAttribute, styles with element.style, and classes with classList to manipulate the elements attributes and styles.
